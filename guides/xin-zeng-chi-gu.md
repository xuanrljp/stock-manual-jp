# 保有株の追加

株式口座の詳細ページに入り、右上の「＋」ボタンをタップし、銘柄コード・購入数量・単価を入力してポジションを作成します。

1. 株式口座の詳細ページに入り、右上の「＋」をタップします。\
   ![list](../.gitbook/assets/stock_account_add_tw.png)
2. 銘柄コード（例：2330）と株式名（例：台積電）を入力します。<br>
3. 購入数量と単価を入力します。<br>
4. 「保存」をタップすると、株式が保有株一覧に表示されます。
