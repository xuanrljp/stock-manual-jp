# 株式資産管理 使用ガイド

> 対応バージョン：8.0 以上

株式資産管理の主な機能：

1. ポジション管理 — 買い・売り・配当取引を記録し、各保有株の状況を簡単に把握
2. 損益レポート — 浮動損益とリターン率を自動計算し、実現損益の詳細レポートも提供
3. 株価自動更新 — 毎日終値後に最新の市場価格を自動取得。台湾株・米国株・日本株・香港株など複数の市場に対応
4. 資産統計 — 株式資産を他の口座と合わせて総合資産概覧に表示
